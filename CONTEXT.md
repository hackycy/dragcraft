# DragCraft Canvas

DragCraft 画布领域描述 schema 节点在画布及嵌套容器中的归属与交互语义。

## Language

**画布根**：
承载所有根级物料的顶层画布范围；它本身不是物料。
_Avoid_: root 节点、根节点

**根级物料**：
由画布根直接拥有的物料，包括 flow、chrome 和 layer placement 下的物料。
_Avoid_: root、根节点、顶层节点

**容器内物料**：
由某个容器区域直接拥有的物料，其空间范围由该物料自身决定。
_Avoid_: 子节点、嵌套节点

**根级区段**：
根级物料在画布根中的完整四边高亮范围，横向不得小于物料自身，纵向边界位于物料外侧，且不随 placement 类型改变；其呈现可以覆盖设备框边界，但不得被相邻 chrome 或内容区遮挡。
_Avoid_: 节点框、物料实际边框

**物料实际范围**：
容器内物料自身的完整空间范围；可被交互安全区裁剪，但裁剪边缘不成为物料的新边界。
_Avoid_: 可见交集、裁剪后节点框

**交互安全区**：
Device Frame 明确提供的内接矩形，用于内容布局避让设备边框、圆角、刘海或其他装饰；它不决定根级区段的高亮范围。
_Avoid_: Device Frame 边界、可见视口、自动推算边距

**交互呈现层**：
Device Frame 提供给画布交互标记的逻辑渲染环境，由 Frame 决定标记与异形边框、装饰和裁剪区域的视觉合成关系；它可以包含多个坐标平面。
_Avoid_: 交互安全区、全局浮层、画布 Portal

**内容平面**：
交互呈现层中随页面内容自然滚动的坐标平面。
_Avoid_: 滚动层、flow 高亮层

**视口平面**：
交互呈现层中相对设备视口保持固定的坐标平面。
_Avoid_: 固定层、chrome/layer 高亮层

**选中高亮**：
标识当前选中物料范围的视觉投影；物料 hover 不生成范围投影。
_Avoid_: hover 框、hover/selected overlay、节点边框

**选中投影**：
当前选中物料的空间范围及其所属坐标平面，与具体颜色、线型或高亮组件无关。
_Avoid_: selected 样式、overlay DOM、outline
